About Me

Nama Saya Fransiskus Sukardi Ruata, saya biasa dipanggil dengan nama Frans, saya dilahirkan pada tanggal 16 Maret 2001 di Rumah Sakit Salak. Saya adalah orang yang biasa yang menyukai kegemaran tentang teknologi, angkasa luar ataupun tentang penemuan-penemuan ilmiah yang baru. Saya dulu bersekolah di:

  • SD NEGERI CILUAR 01
  • SMP NEGERI 15 BOGOR
Saya sekarang bersekolah di SMK - SMAK Bogor, bisa dikata saya memasuki sekolah ini karena:
  • Ayah Saya

    Bisa dikata ayah saya bernama Rudi Sukardi merupakan alumni dari sekolah SMK-SMAK Bogor ini

  • Keinginan Saya

    Keinginan saya yang dulu memang ingin bercita-cita menjadi ilmuwan akhirnya menaikkan minat saya untuk masuk ke sekolah ini dan bisa dikata saya tidak kecewa sama sekali

Bisa dikata saya adalah orang yang pendiam dan tidak banyak berbicara, saya hanya berbicara jika memang sesuai dengan pemikiran saya misalnya dalam bercanda dan lain-lain. Dan juga saya terkadang suka memilih teman, tapi pada akhirnya saya beranggapan bahwa saya bisa berteman dengan siapa saja tetapi saya selalu mengikuti hal-hal yang baiknya saja.

My Hobbies

Saya memiliki banyak sekali hobi, tetapi ada beberapa hobi yang memang paling utama dari saya, yaitu:

  • Bermain Games

    • Bisa dikata saya sangat possesive sekali dengan game, saya menyukai game sejak umur 6 tahun ketika Paman saya memberikan saya console PS1 kepada saya. Saya mulai mencoba beberapa game yang menurut saya sesuai dengan umur saya dan terkadang saya sering bermain bersama ayah saya. Dan akhirnya saya pun pindah ke perangkat PC dan bisa dibilang sudah menghabiskan sekitar Rp.5.000.000 hanya dalam 1 tahun terakhir, menurut orang lain ini sangat boros sekali apalagi dalam hal hobi ini, tapi menurut saya, saya tidak mempermasalahkan hal tersebut karena jika memang sudah hobi nya susah untuk dibendung lagi. Dan saya menyukai hal-hal yang bernama Esport, salah satunya foto diatas dia player kesukaan saya bernama JW di CS:GO, saya berharap bisa menjadi seperti dia kelak nanti.

  • Bermain Bola Basket

    • Hobi ini dimulai ketika saya menginjak kelas 5 SD, bisa dikata di hobi saya ini saya akhirnya memutuskan untuk mencoba masuk ke klub Basket di SMP saya, tetapi saya tidak bertahan lama karena memang jadwalnya ketika di akhir tahun menjelang UN sangat padat sekali sehingga saya memutuskan untuk berhenti dari klub tersebut.

  • Bermain Badminton

    • Hobi ini saya sudah sukai sejak kecil, saya sering terpukau dengan pertandingan-pertandingan badminton ketika ayah saya sedang menonton pertandingan tersebut, akhirnya hingga sekarang ini hobi kedua saya yang masih saya lakukan selain bermain games.

  • Membaca Novel

    • Hobi ini saya lakukan ketika memang tidak ada suatu pekerjaaan dirumah, saya terkadang bosan bermain games dan akhirnya memilih untuk membaca novel sambil beristirahat sejenak dari dunia virtual saya.
